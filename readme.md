📅 South Asian University EventHub

Welcome to the South Asian University EventHub This is a web-based platform designed to make organizing, promoting, and registering for college events as easy as possible. 

Whether you are a student looking to attend a cultural fest, an Event Head organizing a tech hackathon, or an Admin managing the university's activities, this platform has everything you need in one place.

✨ Key Features

The platform is divided into three main roles, each with its own special features:

1. For Students (Users)
Discover Events: Browse through upcoming events sorted by categories (Tech, Sports, Cultural, Academic).
Easy Registration: Sign up for events with just a few clicks.
Stay Updated: See all the details, dates, and locations for the events you care about.

2. For Event Organizers (Event Heads)
Dedicated Dashboard: A private workspace to manage your activities.
Create Events: Easily publish new events with descriptions, dates, times, and registration limits.
Track Registrations: Keep an eye on how many people are signing up for your events.
Manage Profile: Update your department and contact details.
Past Events: Edit , delete and keep an eye on past published events.

3. For Administrators (Admins)
Master Dashboard: Get a bird's-eye view of everything happening on the platform.
Approval System: Review and approve (or reject) events submitted by Event Heads before they go public.
User Management: View and manage all registered students and Event Heads (ban , unban, approve or decline requests of event heads).
Monitor Activity: Keep track of total events, active users, and system login history. 

🛠️ Technologies Used

This project is built using standard web technologies. It is lightweight and runs directly in the browser:
HTML: For the structure and layout of the web pages.
CSS: For styling, responsive design, and making it look great on any device.
JavaScript: For making the website interactive (modals, dashboards, and managing data).

📁 File Structure Overview

•	`index.html` - The main homepage and login area.
•	`admin-dashboard.html` - The management screen for university admins.
•	`event-head-dashboard.html` - The workspace for event organizers.
•	`registrations.html` - The page for all opened registrations.
•	`styles.css` - Contains all the design and layout rules.
•	`app.js` / `admin-dashboard.js` / `event-head.js`/`registrations.js` - The logic that makes the buttons, forms, and dashboards work.
•	`images/` - Folder containing the university logo and event banners.
